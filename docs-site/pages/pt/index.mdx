## Por que CypherLite?

<div className="feature-grid">

<div className="feature-card">
### Zero Configuracao
Abra um arquivo e comece a consultar. Sem processo de servidor, sem instalacao, sem arquivos de configuracao. Apenas um arquivo `.cyl`.
</div>

<div className="feature-card">
### Banco de Dados de Arquivo Unico
Todos os dados do grafo vivem em um arquivo portatil. Copie, faca backup ou compartilhe todo o seu banco de dados de grafos com uma unica operacao de arquivo.
</div>

<div className="feature-card">
### Compativel com ACID
Seguranca transacional completa com Write-Ahead Logging, isolamento de snapshot e recuperacao de falhas. Seus dados sao sempre consistentes.
</div>

<div className="feature-card">
### Biblioteca Embarcada
Vincule o CypherLite diretamente ao seu aplicativo. Disponivel como crate Rust, pacote Python, modulo Go, addon Node.js e biblioteca C.
</div>

<div className="feature-card">
### Sistema de Plugins Extensivel
Quatro tipos de plugins (ScalarFunction, IndexPlugin, Serializer, Trigger) permitem estender o CypherLite conforme suas necessidades.
</div>

<div className="feature-card">
### Consultas Temporais
Consulte seu grafo em qualquer ponto no tempo. O versionamento integrado rastreia cada alteracao em nos, arestas e propriedades.
</div>

</div>

## Exemplo Rapido

```rust
use cypherlite_query::CypherLite;

fn main() -> Result<(), Box<dyn std::error::Error>> {
    let db = CypherLite::open("my_graph.cyl")?;

    db.execute("CREATE (a:Person {name: 'Alice', age: 30})")?;
    db.execute("CREATE (b:Person {name: 'Bob', age: 25})")?;
    db.execute(
        "MATCH (a:Person {name: 'Alice'}), (b:Person {name: 'Bob'}) \
         CREATE (a)-[:KNOWS {since: 2023}]->(b)"
    )?;

    let result = db.execute(
        "MATCH (p:Person) WHERE p.age > 20 RETURN p.name, p.age"
    )?;
    for row in result {
        let row = row?;
        println!("{}: {}", row.get("p.name").unwrap(), row.get("p.age").unwrap());
    }
    Ok(())
}
```

## Destaques de Funcionalidades

| Funcionalidade | Descricao |
|---------|-------------|
| **Property Graph** | Nodes and edges with typed properties (Null, Bool, Int64, Float64, String, Bytes, Array) |
| **Cypher Queries** | openCypher subset with MATCH, CREATE, MERGE, SET, DELETE, RETURN, WHERE |
| **Temporal Versioning** | AT TIME queries, version store, temporal edge versioning |
| **Subgraphs** | Named subgraphs as first-class entities with CREATE/MATCH SNAPSHOT |
| **Hyperedges** | N:M relationships connecting arbitrary numbers of nodes |
| **Plugin System** | ScalarFunction, IndexPlugin, Serializer, Trigger plugin types |
| **FFI Bindings** | C ABI, Python (PyO3), Go (CGo), Node.js (napi-rs) |
