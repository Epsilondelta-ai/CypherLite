# Python 시작하기

CypherLite는 [PyO3](https://pyo3.rs)로 빌드된 Python 바인딩을 제공하며, 그래프 데이터베이스 작업을 위한 네이티브 Python API를 제공합니다.

## 설치

<Steps>

### pip으로 설치

```bash
pip install cypherlite
```

### 설치 확인

```python
import cypherlite
print(cypherlite.__version__)
```

</Steps>

<Callout type="info">
Linux, macOS, Windows용 사전 빌드 wheel이 제공됩니다. 플랫폼에 맞는 wheel이 없는 경우 pip이 소스에서 빌드합니다(Rust 툴체인 필요).
</Callout>

## 기본 사용법

### 데이터베이스 열기

```python
import cypherlite

db = cypherlite.open("my_graph.cyl")
```

### 노드와 관계 생성

```python
db.execute("CREATE (a:Person {name: 'Alice', age: 30})")
db.execute("CREATE (b:Person {name: 'Bob', age: 25})")
db.execute(
    "MATCH (a:Person {name: 'Alice'}), (b:Person {name: 'Bob'}) "
    "CREATE (a)-[:KNOWS {since: 2023}]->(b)"
)
```

### 데이터 조회

```python
result = db.execute("MATCH (n:Person) RETURN n.name, n.age")
for row in result:
    print(f"{row['n.name']} (age: {row['n.age']})")
```

### 업데이트 및 삭제

```python
# Update properties
db.execute("MATCH (p:Person {name: 'Alice'}) SET p.age = 31")

# Delete a relationship
db.execute(
    "MATCH (a:Person {name: 'Alice'})-[r:KNOWS]->(b) DELETE r"
)

# Delete a node
db.execute("MATCH (p:Person {name: 'Bob'}) DELETE p")
```

### 데이터베이스 닫기

```python
db.close()
```

<Callout type="warning">
사용이 끝나면 반드시 `db.close()`를 호출하여 모든 데이터가 디스크에 기록되도록 하세요. 또는 컨텍스트 매니저를 사용할 수 있습니다(지원되는 버전의 경우).
</Callout>

## 소스에서 빌드

모든 기능을 활성화하여 소스에서 빌드하려면:

```bash
cd crates/cypherlite-python
pip install maturin
maturin develop --release
```

## 시스템 요구사항

- **Python**: 3.8+
- **플랫폼**: Linux, macOS, Windows
- **소스 빌드**: Rust 1.84+ 툴체인 필요
