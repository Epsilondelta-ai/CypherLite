## 为什么选择 CypherLite？

<div className="feature-card">

打开文件即可查询。无需服务器进程、无需安装、无需配置文件。只需一个 `.cyl` 文件。

所有图数据存储在一个可移植文件中。通过单次文件操作即可复制、备份或共享整个图数据库。

通过预写日志（WAL）、快照隔离和崩溃恢复提供完整的事务安全。数据始终保持一致。
</div>

<div className="feature-card">

将 CypherLite 直接链接到您的应用程序中。提供 Rust crate、Python 包、Go 模块、Node.js 插件和 C 库。

## 快速示例

```rust
use cypherlite_query::CypherLite;

fn main() -> Result<(), Box<dyn std::error::Error>> {
    let db = CypherLite::open("my_graph.cyl")?;

    db.execute("CREATE (a:Person {name: 'Alice', age: 30})")?;
    db.execute("CREATE (b:Person {name: 'Bob', age: 25})")?;
    db.execute(
        "MATCH (a:Person {name: 'Alice'}), (b:Person {name: 'Bob'}) \
         CREATE (a)-[:KNOWS {since: 2023}]->(b)"
    )?;

    let result = db.execute(
        "MATCH (p:Person) WHERE p.age > 20 RETURN p.name, p.age"
    )?;
    for row in result {
        let row = row?;
        println!("{}: {}", row.get("p.name").unwrap(), row.get("p.age").unwrap());
    }
    Ok(())
}
```

## 功能亮点

| 功能 | 描述 |
|---------|-------------|
| **属性图** | 节点和边支持类型化属性（Null、Bool、Int64、Float64、String、Bytes、Array） |
| **Cypher 查询** | openCypher 子集，支持 MATCH、CREATE、MERGE、SET、DELETE、RETURN、WHERE |
| **时态版本控制** | AT TIME 查询、版本存储、时态边版本控制 |
| **子图** | 命名子图作为一等实体，支持 CREATE/MATCH SNAPSHOT |
| **超边** | N:M 关系，连接任意数量的节点 |
| **插件系统** | ScalarFunction、IndexPlugin、Serializer 和 Trigger 插件类型 |
| **FFI 绑定** | C ABI、Python (PyO3)、Go (CGo)、Node.js (napi-rs) |
